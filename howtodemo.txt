start up: require 3 laptops connect with the same wifi
step 1: Open terminal on each laptop.
python start_sampleapp.py --server-ip 0.0.0.0 --server-port 9000

step 2: Turn on the proxy server in 1 laptop.
python start_proxy.py --server-ip 0.0.0.0 --server-port 8080

Step 3: Connect the Clients
http://10.28.189.138:8080/

Step 4: Login and Auto-Discover

In Tab 1:
Username: a
Password: a

In Tab 2:
Username: b
Password: b

In Tab 3:
Username: c
Password: c

if you want to connect to the web with other device with the same wifi

step 1: turn off the firewall for public network of the host machine

step 2: check the ip of the host device by cmd ipconfig /all and search for IPv4 address in "wireless lan wifi adapter"
